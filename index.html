Extrator de PDF

Programa para extrair dados de documentos PDF

Criei este programa pois um dia vi uma colega minha copiando e-mail por e-mail do sistema da empresa, manualmente, para enviar e-mails em massa, daí tive a ideia de criar um programa que extraísse esses e-mails automaticamente para agilizar seu trabalho. Porém, como não é possível, até onde sei, extrair diretamente do sistema, criei o programa para extrair de documentos PDF, para que elas gerem o PDF ao tentar imprimir e, utilizem meu programa para extrair automaticamente os e-mails ou telefones.
Esta versão para download foi criada apenas em Python, também fiz uma versão web, criada em Flask, porém não encontrei host gratuito para Flask.
Versão: 1.0.1
